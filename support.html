Questions frequentes

Comment inviter mon partenaire ?

Dans l'application, allez dans "Mon couple" puis appuyez sur "Inviter mon partenaire". Vous pouvez partager le lien d'invitation par SMS, email ou toute autre application de messagerie. Votre partenaire devra telecharger l'application et utiliser ce lien pour rejoindre votre espace couple.

Comment modifier mon humeur du jour ?

Depuis le tableau de bord, appuyez sur la carte "Mon humeur" ou sur l'icone d'humeur en haut de l'ecran. Selectionnez l'emoji qui represente le mieux votre etat actuel. Vous pouvez modifier votre humeur autant de fois que vous le souhaitez dans la journee.

Mes donnees sont-elles securisees ?

Oui, la securite de vos donnees est notre priorite. Toutes les communications sont chiffrees (HTTPS/TLS), vos mots de passe sont securises avec bcrypt, et vos donnees sont stockees sur des serveurs securises dans l'Union Europeenne. Pour plus de details, consultez notre politique de confidentialite.

Comment creer un objectif de couple ?

Allez dans l'onglet "Objectifs" et appuyez sur le bouton "+". Definissez un titre pour votre objectif, ajoutez une description optionnelle, et selectionnez une date cible si vous le souhaitez. Vous et votre partenaire pourrez suivre la progression ensemble.

Puis-je utiliser l'application hors ligne ?

L'application necessite une connexion internet pour synchroniser les donnees avec votre partenaire. Cependant, certaines fonctionnalites de base restent accessibles hors ligne, et les modifications seront synchronisees automatiquement des que vous serez reconnecte.

Comment supprimer mon compte ?

Vous pouvez supprimer votre compte depuis les Parametres de l'application ou en suivant les instructions sur notre page Supprimer mon compte. Attention : cette action est irreversible.
